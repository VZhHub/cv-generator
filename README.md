# CV Generator

## Project Overview
CV Generator is a free Multi-Page Application (MPA) built with HTML, CSS, JavaScript and Vite.

The application allows users to create CV (curriculum vitae/resume) and print/export as PDF.

## Features

- Multi-page structure (Index, Personal, Experience, Output)
- Dynamic form handling
- Photo editor functionality
- Resume preview page
- PDF export via print iframe
- Modular JavaScript architecture
- Component-based CSS structure
- Vite build system for production

## Tech Stack

- HTML5
- CSS3 (modular architecture)
- JavaScript (ES Modules)
- Vite

## Build & Run

- npm install
- npm run dev
- npm run build
